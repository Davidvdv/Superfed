  • Hoi. Superfed is een initiatief om cocreatief inhoud van colleges te bepalen rondom frontend development. Cocreatief houdt in dat we samen met jou graag de inhoud willen bepalen. We hopen dat je graag mee wilt doen!

    4. Post links naar onderwerpen via @_superfed
    5. Stel vragen met #qsuperfed via @_superfed
    • Kudo's voor het delen van kennis met anderen

    • Links naar online college materiaal
  • De volgende thema's hebben we al gekozen. De thema's kunnen uiteraard uitgebreid of aangevuld worden. Daarbij hebben we ook een minimale invulling voor de thema's al gegeven. Concrete links naar bronnen staan ook al op twitter via _superfed

HTML5 en CSS3

De basis voor nieuwe ontwikkelingen binnen het domein van frontend development.

Mogelijke Onderwerpen: html5 structuur elementen, canvas, localstorage, css3 transition, mediaqueries.

Twitter hashtag: #htmlcss

Responsive webdesign

Responsive webdesign, ontwikkelen voor meerdere devices

Onderwerpen: mediaqueries, flexible images, flexible lay-out.

Twitter hashtag: #respweb

JavaScript OO

Ontketen de kracht van JavaScript.

Onderwerpen: object oriented, closures, scope van variabelen, prototype.

Twitter hashtag: #jsoop

JavaScript Frameworks en Libraries

Gebruik frameworks en libraries voor het professioneel ontwikkelen van websites

Mogelijke onderwerpen: modernizr, sencha, cappuccino, sproutcore.

Twitter hashtag: #jsframe
