HTML5 features

Op deze pagina zijn een aantal HTML5 features te vinden die ik heb uitgezocht. Ik ben mij ervan bewust dat de ondersteuning van die verschillende features kunnen varieren bij verschillende moderne browsers. Chrome lijkt op dit moment de koploper, met name op het gebied van semantisch tags, zoals: <summery> en <details> bijvoorbeeld. Je moet zoiets als <div class="details"> zo min mogelijk willen, maar juist de tags gebruiken die zeggen welke content het bevat. Net als XML dus. Wat betreft CSS3 wil ik niet gebruiken als gimmicks, maar als waardige toevoeging op de pagina.
